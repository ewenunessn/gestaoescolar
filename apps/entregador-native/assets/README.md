# Assets do App

## Ícones Necessários

Para o app funcionar completamente, você precisa adicionar os seguintes arquivos nesta pasta:

### Obrigatórios:
- `icon.png` - 1024x1024px - Ícone principal do app
- `splash.png` - 1284x2778px - Tela de splash
- `adaptive-icon.png` - 1024x1024px - Ícone adaptativo Android
- `favicon.png` - 48x48px - Favicon para web

## Como Gerar

### Opção 1: Usar Gerador Online
1. Acesse: https://www.appicon.co/
2. Faça upload de uma imagem 1024x1024
3. Baixe o pacote completo
4. Extraia os arquivos para esta pasta

### Opção 2: Usar Imagens Temporárias
Por enquanto, o app vai funcionar sem os ícones. Eles são necessários apenas para o build final.

## Cores do App

- Primária: #1976d2 (azul)
- Secundária: #4caf50 (verde)
- Fundo: #f5f5f5 (cinza claro)
