# 📅 Como Criar um Calendário Letivo

## Passo a Passo Simples

### 1️⃣ Acesse o Calendário Letivo
```
Menu Lateral → Configurações → Calendário Letivo
```

### 2️⃣ Clique no Botão "Novo Calendário"
Na página do calendário, você verá dois botões principais:
- **"Novo Evento"** - Para criar eventos (feriados, recessos, etc.)
- **"Novo Calendário"** - Para criar um calendário de um ano novo ← ESTE AQUI!

### 3️⃣ Preencha o Ano
Um dialog vai abrir pedindo:
- **Ano Letivo**: Digite o ano (ex: 2025, 2026, 2027)
- Você verá a mensagem: "O calendário será criado de 01/02/2025 até 20/12/2025"

### 4️⃣ Clique em "Criar"
Pronto! O calendário foi criado.

## ✅ O Que Acontece Depois?

Após criar o calendário:

1. **Calendário criado** com:
   - Período: 01/02 até 20/12 do ano
   - 200 dias letivos obrigatórios
   - Dias letivos: Segunda a Sexta
   - Status: Ativo

2. **Próximos passos**:
   - Adicione eventos (feriados, recessos, etc.)
   - Ou importe feriados nacionais automaticamente
   - Configure períodos avaliativos (bimestres, trimestres)

---

## 🔄 Diferença entre Calendário e Evento

### 📅 CALENDÁRIO
- É o **ano letivo completo** (ex: 2025)
- Define o período de aulas (01/02 até 20/12)
- Você cria **um calendário por ano**
- Botão: **"Novo Calendário"**

### 📌 EVENTO
- É algo que acontece **dentro do calendário** (ex: Carnaval, Férias)
- Pode ser feriado, recesso, reunião, etc.
- Você cria **vários eventos por calendário**
- Botão: **"Novo Evento"**

---

## 💡 Dicas

### Quando criar um novo calendário?
- No início de cada ano letivo
- Quando precisar planejar o próximo ano
- Para ter calendários de anos anteriores para consulta

### Posso ter vários calendários?
Sim! Um para cada ano (2024, 2025, 2026, etc.)

### Como sei qual calendário está ativo?
O calendário exibido é baseado no **período selecionado** no topo da página.
- Período 2024 → Mostra calendário 2024
- Período 2025 → Mostra calendário 2025

### E se eu criar calendário de um ano que já existe?
O sistema vai dar erro dizendo que já existe calendário para aquele ano.

---

## 🚀 Fluxo Completo

```
1. Criar Calendário (Novo Calendário)
   ↓
2. Adicionar Eventos (Novo Evento)
   - Feriados
   - Recessos
   - Férias
   - Eventos escolares
   ↓
3. Ou Importar Feriados (Menu ⋮)
   - Importa automaticamente feriados nacionais
   ↓
4. Sistema calcula dias letivos automaticamente
   ↓
5. Pronto! Calendário configurado
```

## ❓ Perguntas Frequentes

**P: Onde fica o botão "Novo Calendário"?**
R: Na página Calendário Letivo, ao lado do botão "Novo Evento"

**P: Posso mudar as datas padrão (01/02 até 20/12)?**
R: Sim, mas precisa editar depois de criar (funcionalidade em desenvolvimento) ou criar via script

**P: O que acontece se eu criar calendário 2025 mas estou no período 2024?**
R: O calendário 2025 é criado, mas você só vai vê-lo quando selecionar o período 2025

**P: Posso deletar um calendário?**
R: Sim, mas cuidado! Isso deleta todos os eventos vinculados

**P: Preciso criar calendário para cada escola?**
R: Não! O calendário é único para toda a rede. Cada escola usa o mesmo calendário base.

## 🎯 Resumo Ultra Rápido

1. **Configurações** → **Calendário Letivo**
2. Clique em **"Novo Calendário"**
3. Digite o **ano** (ex: 2025)
4. Clique em **"Criar"**
5. Pronto! ✅

Agora é só adicionar os eventos do ano!
